## 项目运行

```
项目运行之前，请确保系统已经安装以下应用
1、node
2、mongodb 
```

```
git clone https://github.com/zwStar/vue-meituan.git

cd vue-meituan

npm install

npm run dev

访问: http://localhost:8080 (确保后端项目服务已开启)

```
